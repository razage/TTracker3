What is this?

TTracker3 is the 3rd revision of my ticket tracking web application, and this is intended to be the final version. The first two iterations were written in PHP. After learning about Flask, I decided to rewrite the system. I've included new features and I think the project as a whole runs better and is more secure than any of the previous iterations. Integrating Bootstrap also makes it much better looking than any of the other iterations.


How was this made?

TTracker3 was written using CSS, HTML, and Python 3.4. The following frameworks made this possible:

  • Flask
    • Flask-SQLAlchemy
    • Flask-WTF
  • Twitter Bootstrap

You can check out my code on GitHub if you would like to see how the site works. Feel free to submit issues/feature requests, and I'll see what I can do.
